# Release Notes - uxplay-windows v1.2.2

Release date: 2026-03-09

## Overview
`uxplay-windows` remains a tray-first AirPlay receiver setup for Windows based on UxPlay, with Control Center UI, diagnostics, snapshot support, and installer workflow.

## New in v1.2.2

- Added a Control Center footer signature at the bottom-left: `© KaKTools`.
- Updated in-app update target to the new fork releases URL:
  - `https://github.com/kaktools/uxplay-windows/releases`
- Updated GitHub API endpoint for update checks:
  - `https://api.github.com/repos/kaktools/uxplay-windows/releases/latest`
- Updated Help popup README button URL:
  - `https://github.com/kaktools/uxplay-windows/blob/main/README.md`
- Updated tray menu License link to the new repository owner.
- Updated installer metadata URLs (publisher, support, updates) to `kaktools`.
- Updated project documentation (`README.md`) to reflect version `1.2.2` and new fork URLs.
